# Hipotesis General

> Un sistema automatizado de procesamiento y analisis estadistico de imagenes digitales permitira transformar la informacion visual en datos cuantificables y caracterizar el universo de imagenes de la empresa.

## Evidencia hasta ahora

- Pipeline SAM 3 + VLM efectivamente transforma imagenes en datos cuantificables (productos, precios).
- 0% truncamiento JSON en v5 → sistema automatizado y confiable.
- Precision global 50-61% → caracterizacion parcialmente lograda, con margen de mejora.
