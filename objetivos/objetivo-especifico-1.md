# Objetivo Especifico 1 — Seleccion y Visualizacion

> Desarrollar un modulo que permita la seleccion y visualizacion de imagenes digitales.

## Estado

Completado: la webapp Flask permite drag & drop, preview en grilla, y visualizacion de resultados con thumbnails.

## Tecnologias

- Flask 3.1 + HTML5 + Tailwind CSS.
- API de thumbnails via `GET /thumbnail/<job_id>/<file>`.
- Procesamiento con Pillow, OpenCV, PyTorch.
